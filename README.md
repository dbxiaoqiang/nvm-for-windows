# nvm-for-windows
最近在windows开发node程序，感觉windows上没有像linux系统上的nvm这么好用的Package管理器，于是就用winform开发了一个小的工具，用于下载和管理,
node的各个版本的.msi文件
##current function
1、可以获取机器中已经安装的node版本</br>
2、可以下载node各个版本对应的msi文件</br>
3、可以在不同版本的node之间进行切换</br>
4、可以卸载某一个版本的node程序</br>
